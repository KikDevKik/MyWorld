## 2026-01-19 - Localization Consistency in ARIA Labels
**Learning:** UX consistency extends to screen readers. Mixing English ARIA labels in a Spanish-language interface creates a jarring experience for assistive technology users. Always match the `aria-label` language to the application's primary language.
**Action:** When adding accessibility attributes, first verify the UI language (e.g., check `title` attributes or visible text) and translate labels accordingly.
