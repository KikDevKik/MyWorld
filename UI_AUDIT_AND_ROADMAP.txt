1. DIAGNÓSTICO GENERAL
--------------------------------------------------------------------------------
La interfaz actual sufre de "Desconexión Funcional". El backend (Cloud Functions)
es potente y rico en herramientas (análisis, juicios, auditoría), pero el Frontend
no expone estas herramientas de manera accesible.

El problema visual de la `VaultSidebar` (lista cruda) sugiere una falla en la
carga de estilos o un estado "limbo" donde el usuario no ha conectado su Drive,
y la UI no le da una salida clara.

--------------------------------------------------------------------------------

claro o accesible en la UI actual:

3. COMPONENTES "FANTASMA" (EXISTEN PERO NO SE VEN)
--------------------------------------------------------------------------------
* DirectorPanel: Está implementado pero requiere que el usuario sepa cómo abrirlo
  (posiblemente comando de chat o botón oculto).

4. FALLA CRÍTICA DE UX: "EL LIMBO DE CONEXIÓN"
--------------------------------------------------------------------------------
El log `No Cloud Config found` indica que el usuario nuevo entra en un estado
donde la app no sabe qué proyecto cargar.
* ERROR: `VaultSidebar.tsx` recibe la prop `onOpenConnectModal` pero NO LA USA.
  No hay botón para conectar Google Drive. El usuario ve una lista vacía y rota.
* SOLUCIÓN: Se implementará un botón "Conectar Unidad" prominente en la Sidebar.
